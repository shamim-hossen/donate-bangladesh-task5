Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is like a map of a webpage. It represents the structure of the page as a tree, with each element on the page being a node in the tree. JavaScript can use the DOM to access and manipulate different parts of the page, such as changing the text, adding or removing elements, or applying styles. Think of it as a way for JavaScript to interact with and control the content of a webpage.

How do you select an element from the DOM ?

To select an element from the DOM using the following methods
1.getElementById()
2.querySelector()
3.querySelectorAll()

what is event delegation in the context of the dom and why is it usefull ?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements. This means you can attach a single event listener to a parent element, and when an event occurs on any of its child elements, the parent's listener will be triggered.
It is usefull because it can significantly improve performance, especially when dealing with dynamic content or a large number of elements.It can lead to cleaner and more concise code by reducing the number of event listeners.

How do you manipulate an element attributes and styles using the DOM ?

To manipulate an element's attributes and styles using the DOM, you can use methods like setAttribute, getAttribute, and removeAttribute for attributes, and style.property = value, classList.add, classList.remove, and classList.toggle for styles. For example, you can set the href attribute of a link element, get the value of the src attribute of an image element, remove the disabled attribute from a button element, set the background color of an element, add or remove classes to an element, and toggle a class on an element.
